Runge-Kutta de Segundo Orden (Heun)

Esta herramienta permite resolver ecuaciones diferenciales ordinarias utilizando el método de Runge-Kutta de segundo orden (Heun). Ingresa una ecuación, las condiciones iniciales y el valor del paso h para obtener una aproximación numérica de la solución. También puedes incluir una solución exacta si la conoces, para visualizar ambas soluciones de forma gráfica y analizar el comportamiento del error.

Fórmula de Heun (RK2):

k₁ = f(xᵢ, yᵢ)
k₂ = f(xᵢ + h, yᵢ + h·k₁)
yᵢ₊₁ = yᵢ + (h / 2) · (k₁ + k₂)
